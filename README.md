# Immidox - Immigration & Student Consultancy

A modern, responsive web application clone of the Immidox WordPress theme, built with **Next.js 16**, **Tailwind CSS v4**, and **TypeScript**.

## Two Theme Variants

### Theme 1 (Navy + Gold) - `/`
- Fullscreen hero slider, services grid, testimonials, contact form
- **Colors**: Navy blue `#0c4a6e` + Gold `#d4af37`

### Theme 2 (Dark + Coral) - `/theme-2` (matches reference image)
- Top bar with Login, white header, coral red accents
- **Colors**: Dark blue/charcoal `#212844` + Coral red `#F05A4E`
- Sections: Hero, Why Choose Us, About, Quick Help, Stats bar, Team, International CTA, Services bar, Blog, Footer with Instagram feed

## Getting Started

```bash
# Install dependencies (already done)
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- TypeScript
- Lucide React (icons)
